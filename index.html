WOZLLA.js

JavaScript/TypeScript 2D Game Engine

WOZLLA.js is aim to provide completed solution for making HTML5 mobile game. It uses WebGL for 2D rendering, as it want to provide effetive features, it will NOT compatible with Canvas2D in the future.

Features Now

Version 0.5.1 Releaed

  • WebGL Rendering
  • Assets Management
  • Touch Input
  • GameObject and Component
  • Transform and Tween
  • Event System
  • Graphics: Sprite/NinePatch/Mask
  • UI widget: Button/CheckBox
